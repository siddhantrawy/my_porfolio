## 🛠 Tech Stack
- Pure HTML / CSS / JavaScript — zero dependencies
- Responsive & mobile-friendly
- Scroll-reveal animations + animated metric counters
- Terminal-inspired dark UI

## 🚀 Deploy to GitHub Pages (3 steps)

1. Create a new GitHub repository named exactly: `yourusername.github.io`
2. Upload `index.html` to the root of that repository
3. Go to **Settings → Pages → Source → main branch → / (root)** and save

Your portfolio will be live at `https://yourusername.github.io` within minutes.

## ✏️ Customizing
- Open `index.html` in any text editor
- Update contact links, email, LinkedIn URL
- Add your GitHub profile link in the contact section
